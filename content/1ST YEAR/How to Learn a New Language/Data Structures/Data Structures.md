Data Structures (also called Composite Data Types) are values which consist of multiple other values.

They come in a multitude of forms, namely:
- [[Lists]]
- [[Dictionaries]]
- [[Arrays]]
- [[Tuples]]
- [[Files]]
- [[Strings]]
- [[Objects]]
